# Text2img
An encoder and decoder script to convert text to image and back, made for fun
## Requirements
- Python 3.x
- Git
## Getting Started
Follow these steps to get the project running locally:

1. **Clone the repository**
    ```sh
    git clone https://github.com/Thes3us/text2img.git
    cd text2img
    ```
2. **Create a Python virtual environment**
    ```sh
    python3 -m venv venv
    source venv/bin/activate  # macOS/Linux
    venv\Scripts\activate     # Windows
    ```
3. **Install dependencies**
    ```sh
    pip install -r requirements.txt
    ```
4. **Run the encoder**
    ```sh
    python encoder.py
    ```
5. **Run the decoder**
   ```sh
   python decoder.py
   ```
